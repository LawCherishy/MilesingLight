## Engineering realization

The native implementation was developed by Miles using OpenAI Codex for AI-assisted engineering under Miles's direction and decisions. This records process provenance only; OpenAI, Codex, models, and model outputs hold no originating authorship, stewardship, succession, governance, or independent decision authority over MilesingLight. The public source is frozen as seven C++20 source bodies under `src/`.

Each catalog entry states the mathematical tradition from which its finite operation is drawn. These names identify mathematical concepts and do not import another software implementation.

## External concept references

Draw.io/diagrams.net and Godot source trees were studied as concept-space references for interaction, camera, viewport, serialization, ownership, and resource-lifetime knowledge. Those trees, their assets, formats, product identities, and dependency structures are absent from this repository and are not required to build it.

The publication audit compared 2,926 unique Milesing source lines of at least 60 characters with 12,154 Draw.io/Godot reference text and code files and found zero exact long-line matches. This is reproducible engineering evidence, not a substitute for legal advice.

Microsoft documentation informed use of public Win32 and DirectX APIs. The build links only Windows system libraries.

## Deliberate exclusions

This public repository excludes:

- private development and operating instructions;
- internal work-note README files and conversations;
- personal `.mlight` workspaces and machine data;
- Draw.io, Godot, and other reference repositories;
- caches, compiler objects, diagnostics, and temporary recovery bodies.

The Git tree is source-first. The unsigned Windows executable is distributed only as a tagged release asset with a SHA-256 checksum.
