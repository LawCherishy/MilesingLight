DRAW.IO CONCEPT SPACE REBUILT AS ORIGINAL MILESING C++20

STATUS
This is a concept-derived feature map, not a release checklist.  It neither
copies Draw.io source nor claims that every mapped editor feature is present in
v0.2.  REALIZATION.txt and executable verification identify the live subset;
the remaining lines designate original Milesing C++ work to be implemented.

READING RULE
Each line is a feature we understand from the supplied Draw.io source, followed by the original application behavior and the direct code it requires. Reference source, formats, assets, interface, names, and architecture stay in References. Every Light function named below runs inside the same C++ feature on its real typed operands. Where no current Light function fits, the feature uses ordinary original C++ mathematics. Divinand remains with the complete authored work and result.
